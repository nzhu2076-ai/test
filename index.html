📲 Installation Tutorial

Apple iOS (Safari)
  1. Launch the default native Safari Browser to load this workspace.
  2. Tap the official "Share" icon (box containing an upward arrow ⎋) on bottom panel.
  3. Scroll down and tap "Add to Home Screen" option.
  4. Tap "Add" in upper right corner. Start full-screen directly from dashboard!
Android (Chrome)
  1. Launch Google Chrome Mobile Browser to view this page.
  2. Tap the Settings (three dots) menu icon.
  3. Select "Install App" or "Add to Home Screen".
  4. Verify. VoyageEase is now available natively on your phone desk!
